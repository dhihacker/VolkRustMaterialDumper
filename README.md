# VolkRustMaterialDumper

A high-performance utility for scanning the **RustClient.exe** process memory to extract material data.

> **Purpose:** research, debugging, and educational use only.

## Usage
1. Launch `RustClient.exe` (found in your Rust installation directory).
2. Run the scanner executable.
3. Favorited materials are printed to the console as a ready-to-paste C++ array. All materials are written to `output.txt`.
